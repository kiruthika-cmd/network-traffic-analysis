# Network Traffic Analysis Project

This project analyzes network traffic data to identify patterns and anomalies.

## Technologies Used
- Python
- Flask
- HTML, CSS, JavaScript
- CSV dataset

## Features
- Upload network traffic data
- Analyze packets
- Display analysis results via web interface

## How to Run
1. Install Python
2. Install required libraries
3. Run app.py
4. Open browser and access localhost
